üêõ Troubleshooting

Common Issues:

  • 404 on admin pages: Check that static resources are properly configured
  • Login fails: Verify admin user exists and has ADMIN role
  • CORS errors: Check CORS configuration in application.properties
  • File upload fails: Verify file size limits and allowed extensions

Debug Steps:

  1. Open browser developer tools (F12)
  2. Check Console tab for JavaScript errors
  3. Check Network tab for failed API requests
  4. Verify backend logs for server-side errors
